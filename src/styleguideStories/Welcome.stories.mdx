### Tech Overview
- Components are written in React
- The [TailwindCSS](https://tailwindcss.com/) CLI is used to hold theme variables and process class names on the fly at build-time. More on that in [Getting Started](./?path=/docs/getting-started--page).
- [Storybook](https://storybook.js.org/) is used for documentation, testing, and visual display of components. For example, the documentation you are reading right now is a Storybook "story." It's written in MDX using Component Story Format (CSF) -- an ES6 modules-based standard for writing component examples.
