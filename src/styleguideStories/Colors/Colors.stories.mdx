# Colors
#### Here we have a pre-defined color palette, which is used across all of CapCenter's products, both digital (website, apps, etc.) and physical (ads, letterheads, etc.). Most of the time, we'll only be using our 'primary' (Orange 300) and 'secondary' (Green 400) colors, along with an occasional Blue and Slate color.

## How to use
#### The two main ways to apply color to an element are the <code>background-color</code> and <code>text-color</code> CSS attributes. Thankfully, Tailwind gives us an easy way to manipulate these attributes via CSS classes.

- **Text color:** use the `text-{color}` classes, e.g. `.text-primary` or `.text-blue-500`
- **Background color:** use the `bg-{color}` classes, e.g. `.bg-secondary` or `.bg-green-200`

## **Arbitrary Colors**

#### **Need to use an arbitrary, one-off color that doesn't exist in our color library?** No problem! Simply use square brackets to generate a class for that color on-demand, e.g. <code>bg-[#9233ff]</code>. This also works for nearly all utility classes.
