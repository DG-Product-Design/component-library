# Breakpoints

#### We use the following breakpoints across all our applications:
- [XS] Extra Small: 360px
- [SM] Small: 640px
- [MD] Medium: 768px
- [LG] Large: 1024px
- [XL] Extra Large: 1280px
- [2XL] Extra Extra Large: 1440px

<br />

## Try it out
#### Expand your screen width or horizontally scroll to see the different breakpoints below. In order to illustrate the breakpoint sizes, we're using the <code>.max-w-screen-{size}</code> class to limit each element's max-width.
